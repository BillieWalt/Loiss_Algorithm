# Loiss 算法的实现与改进

Loiss 算法是一个面向字节设计的流密码算法． 

该算法整体由两个部分构成: 一个线性反馈移位寄存器 LFSR( linear feedback shift register) ，一个含有 BOMM 结构的过滤函数．

BOMM 结构在抵御代数攻击、线性区分攻击和快速相关攻击方面有良好的性质．

> 流密码 Loiss算法 代数攻击 BOOM结构 LFSR
